M2L Site Statique - Romain MOUROUVIN

Description du projet

Dans ce projet, on est demandé de mettre en place un site statique (Pas de changement dynamique) avec le pure HTML


Les outils utilisés

  • Langage : HTML (Hypertext Markup Language)
  • IDE : Visual Studio Code
  • Outil de Versionning : GitLab (Il faut créer un compte)

Les compétences acquis

  • Répondre aux incidents et aux demandes d’assistance et d’évolution
  • Développer la présence en ligne de l’organisation
  • Travailler en mode projet
